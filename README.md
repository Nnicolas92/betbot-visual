# 🎰 BetBot Visual

Scanner de cuotas deportivas + navegador visual automático.

## Qué hace
- Carga partidos reales de **The Odds API** (40+ casas por partido)
- Calcula el **edge** para cada resultado
- Abre **Chrome visible** y navega a la casa de apuestas
- Guarda **screenshots** de cada paso en `/screenshots/`

## Deportes disponibles
- ⚽ FIFA World Cup (72 partidos ahora mismo)
- ⚽ Copa Libertadores / Sudamericana
- 🏀 NBA / WNBA
- 🎾 Tenis WTA

## Instalación (Windows)
```
doble click → INSTALAR.bat
```

## Uso
```
doble click → CORRER.bat
```
o desde consola:
```
py main.py
```
